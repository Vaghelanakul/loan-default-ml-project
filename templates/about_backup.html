Loan Default Prediction - ML Project

Problem Statement

Predict whether a loan applicant will default on their loan payment based on various demographic and financial features. This helps banks assess risk before approving loans.

Dataset

The dataset contains 255,000+ records with 17 features including Age, Income, Credit Score, Loan Amount, Employment Type, and more financial indicators.
